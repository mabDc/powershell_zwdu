# powershell_zwdu
powershell gui winform 界面 八一中文网 阅读器 https://www.zwdu.com/

## 特点
- zwdu_buttons.ps1使用`button`做目录 zwdu_richBox.ps1使用`RichTextBox`做目录
- powershell 调用 winform
- 界面动态生成
- 界面交互
- `button`点击和`RichTextBox` `Link`点击事件处理
- 网络请求
- gbk解码
- 闭包
- 正则 解析请求内容
- 对象和集合的操作
